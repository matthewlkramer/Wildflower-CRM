# Agent Entry Point

Start with [`replit.md`](replit.md). It is the operating contract for this
repository.

Then, for the subsystem you are changing, follow the reading order in
[`docs/README.md`](docs/README.md).

Selective implementation memory lives in
[`.agents/memory/MEMORY.md`](.agents/memory/MEMORY.md). Read only the entries
relevant to the task; memory is supplemental context, never canonical
architecture.

Authority order, non-negotiable invariants, validation commands, and
documentation rules are defined in `replit.md` and are not duplicated here.
